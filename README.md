This project is a Power BI dashboard built on the Electric Vehicle Population dataset of the United States.
The dashboard analyzes electric vehicle registrations across counties and cities, focusing on vehicle type (BEV/PHEV), manufacturers, model year trends, electric range, and CAFV eligibility.

The purpose of this project is to understand EV adoption patterns, dominant manufacturers, geographic distribution, and technology trends in the U.S.

EV adoption has increased significantly in recent model years (2019 onwards)

Vehicles with higher electric range are mostly BEVs
